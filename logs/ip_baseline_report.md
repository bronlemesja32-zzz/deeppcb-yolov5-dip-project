# Step 6 传统图像处理辅助分析模块报告 (class-agnostic defect proposal baseline)

> 状态: **完成并核验通过, 已停止。** 本模块定位为数字图像处理课程主题的传统方法 baseline
> 与辅助分析工具, 是无类别缺陷候选区域生成(proposal), **不是**完整六类缺陷检测器,
> 不与 YOLOv5s 按 mAP 做直接对比。所有数字来自实际运行输出, 无手填。

## 1. 脚本与输入

- 脚本: `pcb_project/05_ip_baseline.py`
  - 默认模式: 处理 manifest 中 split=test 的 500 张官方测试图
  - `calibrate` 模式: 在 val split(100 图)上标定候选框外扩量 PAD_PX(见 §5)
- 输入:
  - 转换清单: `pcb_project/datasets/deeppcb/manifest.csv`(split=test 的 source_temp_image /
    source_test_image / source_label 三列指向 DeepPCB 原始文件)
  - YOLO test 预测(仅可视化与案例筛选参考): `yolov5/runs/val/deeppcb_yolov5s_50ep_test/labels/`
- 处理样本数: **500**(断言校验); 真值框总数: **3140**(断言校验)
- 运行解释器: DocRescue .venv python 3.12.13(cv2 4.11.0 / numpy 1.26.4), 与 Step 3–5 一致

## 2. 图像处理流程

1. 灰度读取模板图 `_temp.jpg` 与待测图 `_test.jpg`(cv2.IMREAD_GRAYSCALE);
2. 模板差分: `diff = cv2.absdiff(test, temp)`(利用 DeepPCB 模板/待测图已配准的先验);
3. 平滑: 中值滤波 5×5(抑制配准误差产生的细边缘毛刺与椒盐噪声);
4. Otsu 自动阈值二值化(逐图自适应);
5. 形态学: 开运算 3×3 椭圆核(去孤立噪声) → 闭运算 5×5 椭圆核(连接断裂缺陷区域);
6. 连通域分析(8 连通, connectedComponentsWithStats);
7. 候选框过滤: 面积/最小宽高/最大宽高三重过滤(参数见 §5);
8. 候选框外扩 PAD_PX 并裁剪到图像内(对齐 DeepPCB 真值框的"缺陷像素外带边距"标注风格, 见 §5);
9. score = 连通域内平均差分强度 / 255(非统一 1.0, 有实际排序意义), 供贪心匹配排序与输出。

## 3. 评价协议(简单、可读、可审计)

- box-level 贪心匹配: 候选按 score 降序, 依次匹配 IoU 最大且未被占用的真值框,
  IoU ≥ 阈值记 TP; 未匹配候选为 FP; 未匹配真值为 FN。每个真值框至多匹配一次。
- P = TP/(TP+FP), R = TP/(TP+FN), F1 = 2PR/(P+R)(分母为 0 时记 0)。
- 双阈值: **IoU=0.33**(DeepPCB 官方评估脚本默认 IoU constraint, 宽松定位参考)与
  **IoU=0.50**(常见目标检测阈值)。不调用官方 Python2 评估脚本。
- 匹配为 class-agnostic: 只看定位, 不比类别。

## 4. 与 YOLOv5s 的关系(表述口径)

- 传统方法: **class-agnostic defect proposal**(无类别候选区域生成);
- YOLOv5s: **six-class object detection**(六类检测, test 指标见 test_eval_report.md:
  P 0.951 / R 0.944 / mAP50 0.968 / mAP50-95 0.594);
- 两者协议不同(类别无关 vs 类别相关; 简单贪心匹配 vs PR 曲线/mAP), **不能直接比较**,
  只在定位能力与可视化层面讨论。可视化中的 YOLO 框取 conf≥0.25(仅展示/案例筛选用);
  脚本对 YOLO 框做的同款 class-agnostic 匹配也仅用于失败案例自动筛选, 非正式指标。

### PAD_PX 的来历(诚实记录)

初版(无外扩)在 test 上 IoU=0.33 F1 仅 **0.0474**。可视化诊断发现候选框全部紧贴缺陷像素,
而 DeepPCB 真值框在缺陷外带明显边距(真值框均值 38.9×35.7 px, 小缺陷框到 18×19 px),
小框对大框的 IoU 天花板低于 0.33 —— 是**框尺寸约定不匹配**, 不是没找到缺陷。
处理: 在 **val split(100 图)** 上扫描 PAD 候选并冻结最优值, **不在 test 上调参**:

| pad | P/R/F1 @0.33 | P/R/F1 @0.50 |
|---|---|---|
| 0 | 0.0614 / 0.0623 / 0.0619 | 0.0058 / 0.0059 / 0.0059 |
| 4 | 0.5526 / 0.5608 / 0.5567 | 0.1330 / 0.1350 / 0.1340 |
| 6 | 0.6857 / 0.6958 / 0.6907 | 0.4942 / 0.5015 / 0.4978 |
| 8 | 0.6944 / 0.7047 / 0.6996 | 0.6608 / 0.6706 / 0.6657 |
| **10** | **0.6974 / 0.7077 / 0.7025** | **0.6827 / 0.6929 / 0.6878** |
| 12 | 0.6944 / 0.7047 / 0.6996 | 0.6813 / 0.6914 / 0.6863 |
| 14 | 0.6944 / 0.7047 / 0.6996 | 0.6769 / 0.6869 / 0.6819 |
| 16 | 0.6915 / 0.7018 / 0.6966 | 0.6272 / 0.6365 / 0.6318 |

pad=10 在两个阈值下均最优, 且 8–14 为平台区(选择稳健)。冻结 PAD_PX=10 后运行 test。

## 8. 成功案例总结(数字来自 per-image csv)

- **70/500** 张图 F1@0.33 = 1.0(完美检出, 如 20085311, 00041201, 00041205, 00041208);
  **216/500** 张 ≥ 0.8; **414/500** 张 ≥ 0.5; 逐图 F1 中位数 0.75, 均值 0.706。

  group77000 0.940, group00041 0.903, group50600 0.848 —— 差分信号干净时,
  "差分+Otsu+形态学"几乎不出错。
- 附带发现: 传统方法共找到 **49 个** YOLO(conf≥0.25)漏检的真值框(分布在 43 张图),
  例如 90100033(3 个)、20085311(2 个) —— 简单先验方法对深度模型有小而真实的互补性。

## 9. 失败案例总结(可视化在 failure_cases/, 文件名=类型_图号)

1. **FP 爆炸型**(fp_*): 90100028(FP=59)、90100027(56)、12100009(46)、12100007(41)。
   机理: 高密度细点阵/细走线板 + 模板与待测图 1–2 px 配准残差 → 全板结构边缘出现差分晕环,
   开运算无法全部去除, 存活为大量中等尺寸噪声连通域。FP 高度集中:
   group90100+12100+12300 三组贡献 1075/1194 ≈ **90%** 的 FP(前两组即占 63.2%),
   其余 8 个 group 中 3 个 FP 为 0。
2. **FN 型**(fn_*): 20085310(FN=8)、44000013(7)、90100017(7)、20085298(6)。
   机理: 细小/弱对比缺陷的差分响应过小, 被 MIN_AREA/MIN_WH 过滤或 Otsu 全局阈值淹没 ——
   为压制 FP 而设的过滤门槛必然牺牲小目标, 是固定阈值流水线的固有精度-召回张力。
3. **YOLO 好而传统失败型**(yolo_ok_ip_fail_*): 恰为 4 张零候选图(20085298 等),
   YOLO 六类全中(yolo_fn=0)而传统方法颗粒无收 —— 学习式方法对弱信号缺陷的外观建模优势明显。
4. **传统找到而 YOLO 漏检型**(ip_found_yolo_miss_*): 90100033(ipOnly=3)、20085311(2)、
   44000031(2)、44000045(2)。该类型存在(共 49 框), 说明差分先验能补充少量深度模型盲区。

## 10. 可写入课程报告的结论(表述克制)

1. 传统差分方法充分利用了 DeepPCB "模板-待测图已配准"的先验, 在布局稀疏、差分信号干净的
   板面上非常有效(多个 group 平均 F1 0.85–1.00);
2. 它对明显外观差异敏感, 定位精确(IoU 0.33→0.50 指标几乎不降);
3. 但对细小缺陷、弱对比缺陷不稳定(FN 主因), 对配准残差与高密度纹理噪声敏感(FP 主因,
   90% 的 FP 集中于三个细点阵 group);
4. YOLOv5s 在学习缺陷外观与类别语义方面明显更强(且能分六类, 传统方法不能);
5. 传统图像处理仍有价值: 作为无类别候选区域生成器(class-agnostic proposal)、
   可解释的诊断工具(diff/Otsu/形态学中间结果全程可视化), 并对深度模型有小幅互补
   (49 个 YOLO@0.25 漏检框被差分法找到)。
